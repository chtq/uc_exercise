# 部署

##安装xblock和重启
```
$sudo -u edxapp /edx/bin/pip.edxapp install ./myxblock
$sudo /edx/bin/supervisorctl -c /edx/etc/supervisord.conf restart edxapp:
```

##启动服务端
```
$./1.py
```
config.json是配置文件
